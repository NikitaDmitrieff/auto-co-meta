# Projects Directory

This directory contains all projects built by the auto-co AI team.

## Convention

Each project lives in its own subdirectory:

```
projects/
  my-project/          # project root
    README.md          # what it is, how to run it
    ...                # source code, configs, etc.
```

## Rules

- **All new projects must be created inside `projects/`** — never in the repo root.
- Each project should have its own `README.md` with: what it is, how to start it, and where it's deployed.
- Projects are gitignored by default (`projects/*` in root `.gitignore`). To track a project's source in this repo, add an explicit exception: `!projects/my-project/`.
- Deployed URLs and service IDs should be recorded in the project's README and in `memories/consensus.md`.
